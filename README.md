# Last Read Marker

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A Chrome extension that saves your last read position for articles and lets you jump back instantly.

## Features
- Save the current page and scroll position.
- Keep a list of saved pages, open and auto-scroll to the last position.
- Syncs across Chrome devices via Chrome Sync.

## Install (developer mode)
1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click **Load unpacked** and select this folder.

## Usage
1. Open an article and scroll to where you want to stop.
2. Click the extension and choose **Save current page**.
3. Later, open the extension and click **Open & jump**.

## Notes
- Sync requires Chrome sign-in with sync enabled.
- For production use, package and publish to the Chrome Web Store.
